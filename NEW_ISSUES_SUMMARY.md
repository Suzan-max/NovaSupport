## 🚀 Next Steps for Contributors

### Getting Started:
1. Browse issues at: https://github.com/OlaGreat/NovaSupport/issues
2. Filter by labels:
   - `frontend` - React/Next.js work
   - `backend` - Node.js/Express/Prisma work
   - `contract` - Rust/Soroban work
   - `enhancement` - Improvements to existing features
   - `feature` - New functionality
3. Pick an issue that matches your skill level
4. Comment on the issue to claim it
5. Fork the repo and start coding!

### Issue Labels Guide:
- **frontend** - Frontend/UI work
- **backend** - Backend/API work
- **contract** - Smart contract work
- **enhancement** - Improvements to existing features
- **feature** - New functionality
- **easy** - 1-3 hours, good for beginners
- **medium** - 4-7 hours, intermediate level
- **hard** - 8-15 hours, advanced level
